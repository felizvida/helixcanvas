# HelixCanvas v0.3.0

HelixCanvas `v0.3.0` is the first release centered on the **personal biomedical figure workstation** direction.

## Highlights

- richer text and connector editing with activation, inhibition, elbow routes, labels, and stronger typography controls
- AI-native figure editing with a command palette and structured scene edits applied to the real canvas model
- snapshot comparison and local figure branching for safer iterative revision
- semantic asset retrieval and domain starter kits for oncology, immunology, neuroscience, and microscopy workflows
- installable offline-ready app shell with service worker caching and an offline fallback page

## Included In This Release

- new connector and figure-style helpers in `src/lib/connectors.js` and `src/lib/figureStyles.js`
- snapshot diff summaries in `src/lib/projectCompare.js`
- domain starter kits in `src/data/domainStarterKits.js`
- service worker, web manifest, offline page, and installable app metadata under `public/`
- refreshed README and personal-workbench direction docs to match the shipped product

## Validation

The release was validated with:

```bash
npm run check
```

## Release Intent

This cut is meant to make HelixCanvas feel meaningfully more complete for one researcher working locally:

- safer iteration through compare-and-branch workflows
- faster figure starts through domain kits and semantic retrieval
- stronger daily-driver editing through richer text, connectors, and command-palette actions
- better resilience through installable offline-first behavior
